# cv-object-detection-project
# Real-Time Object Detection using YOLOv8

## Problem Statement
In real life, identifying objects quickly is important in areas like surveillance, automation, and assistive technology.

## Solution
This project uses a pre-trained YOLOv8 model to detect objects in real-time using a webcam.

## Features
- Real-time object detection
- Uses deep learning (YOLOv8)
- Simple and efficient

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## Installation
pip install -r requirements.txt

## Run
python main.py

## Output
The webcam opens and detects objects with bounding boxes.

## Learning Outcomes
- Learned object detection concepts
- Understood real-time CV pipeline
- Worked with pre-trained models
